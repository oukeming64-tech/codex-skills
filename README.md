# Codex Skills

Personal Codex skills distilled from real project work.

## Skills

- `handoff-auditor`: audit implementation handoffs before accepting, merging, or declaring them complete.
- `docs-sync-guardian`: keep repository documentation aligned with code, configuration, asset, release, workflow, or product changes.

## Local Use

Copy a skill folder from `skills/` into `~/.codex/skills/`.

Current local copies are installed at:

- `~/.codex/skills/handoff-auditor`
- `~/.codex/skills/docs-sync-guardian`
